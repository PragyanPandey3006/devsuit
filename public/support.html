Frequently Asked Questions

How do I authenticate with GitHub?

Click the "Continue with GitHub" button on the login page. You'll be redirected to GitHub to authorize DevSuit. Make sure you have the required permissions: repo, user, and read:org.

What data does DevSuit access?

DevSuit only accesses repository data, user profile information, and organization membership. We never store your data without permission and only use it to provide analytics and insights.

How do I set up webhooks for auto-labeling?

Go to your repository settings, navigate to Webhooks, and add a new webhook with the URL provided in the AutoLabeler module. Select "Issues" and "Pull requests" events.

Is DevSuit free to use?

Yes, DevSuit is completely free to use. It's an open-source project designed to help developers better understand and manage their GitHub repositories.

Can I contribute to DevSuit?

Absolutely! DevSuit is open source. Check out our GitHub repository and read the contributing guidelines to get started.
