Privacy Policy

This Privacy Policy explains how DevSuit collects, uses, and protects your information when you use our GitHub developer dashboard service.

Table of Contents

1. Information We Collect

GitHub Account Information

When you authenticate with GitHub, we receive the following information:

  • GitHub username and display name
  • Email address associated with your GitHub account
  • Profile picture/avatar URL
  • Public profile information (bio, location, website)
  • Repository access permissions you grant

Repository Data

We access repository data through GitHub's API to provide our services:

  • Repository metadata (name, description, language, stars, forks)
  • Contributor information and statistics
  • Issue and pull request data
  • Commit history and activity
  • Repository health indicators (README, license, etc.)

Usage Information

We collect information about how you use DevSuit:

  • Pages visited and features used
  • Time spent on the application
  • Browser type and version
  • Device information and screen resolution
  • IP address and general location

2. How We Use Your Information

Service Provision

We use your information to:

  • Provide repository analytics and visualizations
  • Generate health scores and recommendations
  • Enable auto-labeling and automation features
  • Display contributor insights and activity data
  • Facilitate API testing and development tools

Service Improvement

We may use aggregated, anonymized data to:

  • Improve application performance and user experience
  • Develop new features and functionality
  • Analyze usage patterns and trends
  • Fix bugs and technical issues

3. Data Storage and Security

🔒 Local Storage Only

Important: DevSuit stores your GitHub access token and session data locally in your browser only. We do not store your personal data or repository information on our servers.

  • Access tokens are stored in browser localStorage
  • No personal data is transmitted to our servers
  • All repository data is fetched directly from GitHub APIs
  • Session data is cleared when you log out

Security Measures

  • HTTPS encryption for all data transmission
  • OAuth 2.0 secure authentication flow
  • Webhook signature verification
  • Regular security updates and monitoring
  • Minimal data collection principle

4. GitHub Integration

OAuth Permissions

DevSuit requests the following GitHub permissions:

repo

Access to repository data for analytics and health scoring

user

Read user profile information for dashboard personalization

read:org

Read organization membership for organization analytics

⚠️ Data Access

DevSuit only accesses data necessary for providing our services. We do not access or store:

  • Repository source code content
  • Private communications or messages
  • Sensitive security information
  • Financial or billing information

5. Data Sharing and Disclosure

No Data Sharing

We do not sell, rent, or share your personal information with third parties.

We may disclose information only in the following limited circumstances:

  • When required by law or legal process
  • To protect our rights, property, or safety
  • To prevent fraud or security threats
  • With your explicit consent

6. Your Rights and Choices

Access and Control

You have the right to:

  • Access your personal information
  • Correct inaccurate information
  • Delete your account and data
  • Revoke GitHub OAuth permissions
  • Export your data

Data Deletion

To delete your data:

  1. Log out of DevSuit (clears local storage)
  2. Revoke OAuth permissions in your GitHub settings
  3. Clear your browser data and cookies
  4. Contact us for complete data removal

7. Cookies and Tracking

Minimal Cookie Usage

DevSuit uses minimal cookies and local storage for:

  • Authentication state management
  • User preferences and settings
  • Session management
  • Basic analytics (anonymized)

You can disable cookies in your browser settings, but this may affect application functionality.

8. Children's Privacy

DevSuit is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13. If you are a parent or guardian and believe your child has provided us with personal information, please contact us.

GitHub's Terms of Service require users to be at least 13 years old, and DevSuit follows the same age requirement.

9. Changes to This Policy

We may update this Privacy Policy from time to time. When we do, we will:

  • Post the updated policy on this page
  • Update the "Last Updated" date
  • Notify users of significant changes via email or in-app notification
  • Maintain previous versions for reference

Your continued use of DevSuit after changes constitutes acceptance of the updated policy.
